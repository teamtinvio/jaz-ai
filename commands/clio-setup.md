# Set up Clio Kit / add a company

Follow the **setup** flow in the `clio-kit` skill (`references/flows.md`). Onboarding questions are in `references/org-interview.md`; file scaffolds are in `references/templates.md`.

## Usage

```
/clio-setup                 first run — creates the kit and connects a company
/clio-setup Acme Pte Ltd    add another company
```

Run it as often as you have companies. It detects an existing kit and goes straight to adding the next one.
