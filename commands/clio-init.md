# Set up Clio Kit / add an organization

Follow the **init** flow in the `clio-kit` skill (`references/flows.md`). Field-by-field onboarding guidance is in `references/org-interview.md`; file scaffolds are in `references/templates.md`.

## Usage

```
/clio-init
/clio-init Acme Pte Ltd
```

Run it again any time to connect another organization — it detects an existing kit and goes straight to onboarding.
